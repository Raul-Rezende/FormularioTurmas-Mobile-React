# Checkpoint 01 - Mobile Application Development 📱

Este projeto é a entrega do Checkpoint 01 da disciplina de Mobile Application Development, do curso de Tecnologia em Desenvolvimento de Sistemas (2TDSPF) da FIAP. O aplicativo foi desenvolvido em React Native e consiste em um formulário de cadastro simples.

---

## 🎯 Objetivo do Projeto
Desenvolver um aplicativo que permite ao usuário preencher informações em um formulário e visualizar os dados inseridos na própria tela após o envio.

## ⚙️ Funcionalidades e Requisitos Atendidos
O aplicativo cumpre todos os requisitos estruturais solicitados:
* **Interface:** Utilização dos componentes `SafeAreaView`, `View` e `Text` para organizar a tela.
* **Formulário:** Campos de entrada para Nome, Curso, Disciplina e Descrição usando `TextInput` com `placeholder`.
* **Interação:** Uso do componente `Button` para o envio dos dados.
* **Gerenciamento de Estado:** Controle de cada campo do formuláriO utilizando o Hook `useState`.
* **Ciclo de Vida:** Utilização do Hook `useEffect` para registrar uma mensagem no console quando o aplicativo é iniciado.
* **Exibição:** Apresentação dos dados preenchidos organizados na tela após o envio.
* **Estilização:** Aplicação de `StyleSheet` para definir tamanhos de texto, margens e organização visual.
