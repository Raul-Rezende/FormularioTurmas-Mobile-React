# Checkpoint 01 - Mobile Application Development 📱

[cite_start]Este projeto é a entrega do Checkpoint 01 da disciplina de Mobile Application Development, do curso de Tecnologia em Desenvolvimento de Sistemas (2TDS) da FIAP[cite: 1, 2, 3]. [cite_start]O aplicativo foi desenvolvido em React Native e consiste em um formulário de cadastro simples[cite: 4].

[cite_start]👨‍🏫 **Professor:** Fernando Pinéo [cite: 2]

---

## 🎯 Objetivo do Projeto
[cite_start]Desenvolver um aplicativo que permite ao usuário preencher informações em um formulário e visualizar os dados inseridos na própria tela após o envio[cite: 5].

## ⚙️ Funcionalidades e Requisitos Atendidos
[cite_start]O aplicativo cumpre todos os requisitos estruturais solicitados[cite: 6]:
* [cite_start]**Interface:** Utilização dos componentes `SafeAreaView`, `View` e `Text` para organizar a tela[cite: 9, 10, 11].
* [cite_start]**Formulário:** Campos de entrada para Nome, Curso, Disciplina e Descrição usando `TextInput` com `placeholder`[cite: 14, 15, 16, 17, 19, 20].
* [cite_start]**Interação:** Uso do componente `Button` para o envio dos dados[cite: 21, 22, 23].
* [cite_start]**Gerenciamento de Estado:** Controle de cada campo do formulário utilizando o Hook `useState`[cite: 25, 26].
* [cite_start]**Ciclo de Vida:** Utilização do Hook `useEffect` para registrar uma mensagem no console quando o aplicativo é iniciado[cite: 40, 41, 44].
* [cite_start]**Exibição:** Apresentação dos dados preenchidos organizados na tela após o envio[cite: 34, 39].
* [cite_start]**Estilização:** Aplicação de `StyleSheet` para definir tamanhos de texto, margens e organização visual[cite: 46, 47, 48].
